# Quiz Game

## Как запустить
```
npm install
npm start
```

- Админка: `/evergameadmin865`
- Вопросы в `questionbase.json`.
- Кнопки: Open, Close, Start.
- Игра: `/`.
  1. Введите ник.
  2. Лобби ждёт других.
  3. Админ нажимает Start → обратный отсчёт.
  4. Вопросы, подсветка ответов.
  5. Победа первого, вывод статистики.
